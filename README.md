# Calculator - React - [Live](https://calculator-react-milos.herokuapp.com//)

Calculator created for practice using React framework.

## Requirements

- Node.js

- React

- React-DOM

- React-Create-App

- npm

- CSS

- ES6

## Installing

- npm install

## Available Scripts

- `npm start`

- `npm test`

- `npm run build`

- `npm run eject`
