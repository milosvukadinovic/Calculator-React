## React calculator

This is a React Calculator project.

## Requirements

- Node.js

- React

- React-DOM

- React-Create-App

- npm

- CSS

- ES6

## Installation

Run `npm install`

## Available Scripts

- `npm start`

- `npm test`

- `npm run build`

- `npm run eject`
